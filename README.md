## ✨ About ClientReplyAI

**ClientReplyAI** is a premium low-budget AI SaaS web app built with Laravel, Livewire, Tailwind CSS, Gemini, and Groq.

It helps users write better replies in seconds by converting rough text into professional, polite, confident, and ready-to-send messages.

---

## 🎯 Core Features

- 🤖 AI-powered professional reply generation
- 💬 ChatGPT-style reply workspace
- ⚡ Quick Reply Mode
- 🧠 Advanced Mode with tone, use case, language, context, and goal
- 🌍 Roman Urdu / Urdu / Hindi / broken English to professional English
- 📌 Saved replies
- ⭐ Favorite replies
- 🕘 Chat history grouped by date
- 🧩 Ready-made templates
- 👤 Writing preferences profile
- 📊 Free-plan daily usage limit
- 🔁 Gemini primary + Groq fallback provider system
- 🔒 API keys stored only in `.env`
- 📱 Mobile-responsive premium UI
